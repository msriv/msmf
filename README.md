# MSMF Website

Redesigned version of MSMF website.

To run only the client side -

```
$ cd ./v2client
$ yarn
$ yarn dev
```

To only run the server -

```
$ cd ./v2server
$ php -S localhost:8000
```

To run the complete build locally, we have scaffolded a docker setup. You can run it by doing

```
$ docker-compose up
```
